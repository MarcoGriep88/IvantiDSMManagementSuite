Was ist die DSM Management Suite?

Die DSM Management Suite ist eine unabhängige Erweiterung für die Ivanti DSM Infrastruktur und hat den Fokus, für das Software- und Patch Management, detailliertere Informationen und Reports zu liefern. Die DSM Management Suite sammelt dabei in regelmäßigen abständen Informationen aus Ihrer Ivanti DSM Umgebung (z.B. zu Computer-Sicherheitslücken) und sendet diese Informationen an die Web API von DSM Management Suite. Das Windows Dashboard ermöglicht Ihnen auf diese Daten zuzugreifen und Auswertungen zu erstellen (häufigsten Sicherheitslücken, Aktueller Patch Stand, Historie und Trendlinie). Im Gegensatz zu XTraction liegt der Fokus auf Patch Informationen. Da die Software regelmäßig (täglich, wöchentlich, monatlich) alle Sicherheitslücken von Computerobjekten erfasst, erhalten Sie einen Historie. Dadurch sehen Sie wie effektiv Sie Ihre Umgebung Patchen. Oft sind Unternehmen zu konservativ was das Freigeben von Updates angeht und es kommen immer mehr Sicherheitslücken hinzu als gefixt werden. Das Backend (Web API) ermöglicht Ihnen, mit jeder beliebigen Programmiersprache (z.B. Python) alle Informationen vom DSM Management Suite abzurufen und selbst zu analysieren, sofern notwendig.

Die wichtigsten Features

Erweitertes Patch Management Ziehen Sie aussagekräftige Patch Reports aus Ihrer Umgebung. Identifizieren Sie welche Third-Party Applikation die meisten Lücken aufweist und wie effektiv tatsächlich Ihr Unternehmen geschützt ist.

Historische Daten DSM Management Suite zeigt Ihnen einen Patch Verlauf an mit dem Sie prognosen für die Zukunft erstellen können. Schließen Sie mehr Sicherheitslücken als hinzukommen?

Manager Dashboard Das Windows Dashboard wie auch das Web Dashboard sind optimal für IT-Manager. Sie brauchen sich in keine komplizierte DSM-Konsole einzuarbeiten.

Mächtige Web API Alle Patch Management Daten sind über eine performante Web-API auswertbar. Nutzen Sie Ihre favorisierte Programmiersprache wie PowerShell oder Python um individuelle Reports oder Automatisierungsaufgaben zu entwickeln.
