Das bietet die DSM Management Suite

Die wichtigsten Features

Erweitertes Patch Management Ziehen Sie aussagekräftige Patch Reports aus Ihrer Umgebung. Identifizieren Sie welche Third-Party Applikation die meisten Lücken aufweist und wie effektiv tatsächlich Ihr Unternehmen geschützt ist.

Historische Daten DSM Management Suite zeigt Ihnen einen Patch Verlauf an mit dem Sie prognosen für die Zukunft erstellen können. Schließen Sie mehr Sicherheitslücken als hinzukommen?

Manager Dashboard Das Windows Dashboard wie auch das Web Dashboard sind optimal für IT-Manager. Sie brauchen sich in keine komplizierte DSM-Konsole einzuarbeiten.

Mächtige Web API Alle Patch Management Daten sind über eine performante Web-API auswertbar. Nutzen Sie Ihre favorisierte Programmiersprache wie PowerShell oder Python um individuelle Reports oder Automatisierungsaufgaben zu entwickeln.
